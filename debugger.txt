Setup Debugger For JavaScript
-----------------------------

   - Two Way Of Debug 
                  |
                  |---------------- Web Application 
                  |
                  |---------------- Node Js CLI 


Step - 1
--------
  - Search on Top Bar for Vs-Code (launch.json)

  - Add below code to launch.json file for Web Application Debug 
   
            {
                "version": "0.2.0",
                "configurations": [
                    {
                    "name": "Attach to Chrome",
                    "port": 9222,
                    "request": "attach",
                    "type": "chrome",
                    "webRoot": "${workspaceFolder}"
                    }
                ]
            }

  - Add below code to launch.json file for Node Js CLI   Debug (need to be inside the configurations [] like above )

              {
                "name": "Debug CLI Quiz App (Node.js)",
                "type": "node",
                "request": "launch",
                "program": "${workspaceFolder}/1-frontend/Javascript-GFG/projects/cli-quiz-app/script.js",
                "console": "integratedTerminal"
              }


Step -2
-------

       - Top right Corner you can "Run" 

       - Click on Run ---------> Start Debugging 

       - You Can See at top-right corner (RUN AND DEBUG) options 
                                                |
                                                |----------- Select the Respective Environment (e.x - Debug CLI Quiz App (Node.js)- this we given at launch.json)


       - Before Start Set the break-point (to the file )      

       - Click On The Green Right Error Button to start the Debugging
